## Instruction ファイル作成
```
以下の理由から、GitHub Copilot Instructionファイルを作成したいです。
手伝ってください。

VS Code で GitHub Copilot を利用して、AI駆動で、自動テストのコードを作りたいです。
自動テストのフレームワークは、Playwrightを使います。
言語はTypeScriptです。
POMなどの設計原則も決めておきたいです。

５名ほどのチームで、自動テストコードを作成します。
複数人開発ということもあり、注意事項もあれば教えてください。

自動テストコードのリポジトリは、GitHubに作成予定です。

テスト仕様書は、EXCEL形式なのですが、.mdファイルにしたいと思っています。
テスト仕様書は２種類あります。
　リグレッションテスト（１シートのみ）
　結合テスト（複数シートあり）

テスト対象のシステムは、Webシステムです。
ステージング環境で実施します。
CIツールとしてGitHub Actionsを使います。
```
